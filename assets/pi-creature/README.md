# Matrix Pi Creature

These SVGs are local matrix-colored derivatives of `PiCreatures_plain.svg` from the `cairo-backend` branch of `3b1b/manim`:

`manimlib/files/PiCreatures_plain.svg`

The original source is MIT licensed by 3Blue1Brown LLC. The copied license text is in `LICENSE-3B1B-MANIM.txt`.

The current `3b1b/3Blue1Brown.com` repository also publishes `public/images/pi-creatures/*.svg`, but that repository marks media in `public` as all-rights-reserved. Those files are not copied here.

`CaftBotti/manim_pi_creatures` was checked as a reference, but its README says the `PiCreature` folder is sourced from `www.3blue1brown.com/images/pi-creatures/*.svg` and asks not to use it commercially. The homepage mascot therefore uses the MIT Manim base plus local expression and matrix detailing instead of copying those files.
