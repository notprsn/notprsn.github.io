Love Letters

An encrypted archive.

This page ships ciphertext only. Decryption happens in your browser after you enter the passphrase. The key is not stored in the repository, and the plaintext never needs to be published.

Security model

Public GitHub Pages cannot give perfect secrecy if a browser must eventually decrypt the content. The practical defense is strong encryption, no committed password, and a long passphrase shared off-platform.

The archive is locked. If the ciphertext bundle has not been initialized yet, this page will tell you.

Public details

  • PBKDF2-SHA-256 for key derivation.
  • AES-256-GCM for authenticated encryption.
  • Random salt and IV for each generated bundle.
  • Password never committed to the repository.
