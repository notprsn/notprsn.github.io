Copy this directory, rename it, and keep the HTML, CSS, and JS local to the new experiment.

  1. Duplicate `fun/projects/template/` into a new slug.
  2. Keep page-specific assets inside that new directory.
  3. Add one card on `/fun/` linking to the new project.
