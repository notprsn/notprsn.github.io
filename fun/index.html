Fun Stuff

A lab for small visual and mathematical toys.

This section is now structured as a set of self-contained mini projects. Each experiment should live in its own subdirectory with its own HTML, CSS, and JS.

Directory rule

Create new pieces as their own local subdirectories so starting, iterating, and finishing one project never breaks the rest.
